How to compile the code
1. Download the file to a linux machine
2. Untar the file
3. In the working directory, run the following command: make

How to execute the code
1. In the working directory, run the following command: ./program
2. Various commands are available to you, with provided descriptions
3. Typing in either of the following commands will cycle between the parts of the assignment: next; prev
4. Type end, exit, or quit to close the program (or just ctrl+c, it won't matter)

Notes
1. Graph view indicates the local immediate links between nodes. Blanks indicate there is no immediate link between those two nodes.
2. Algorithm view indicates the length of the shortest path between any two nodes
